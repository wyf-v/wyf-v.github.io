Typora是一款功能强大的Markdown编写工具，它可以不用分屏也能实时预览效果，对于引用的图片可以一键转为本地，或上传到自己的服务器上，并自动替换图片的链接。

我有一个阿里云的服务器，虽然只有1M的小水管，但是没有额外限制，用来当个图片服务器还是可行的。

Typora支持多种图片上传服务

我选择自定义命令（Custom Command）

在其Upload-Image文档中描述了将会在命令行后面添加多个图片路径参数，并且可以自动将返回结果里的url提取出来，并替换文章中的链接。
